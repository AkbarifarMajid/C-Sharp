# RätselSpiel (BrainTech Games Prototype)

Ein Konsolenspiel in C#, das dem Benutzer mathematische und logische Rätsel stellt. Der Benutzer sammelt Punkte für korrekte Antworten.

## Features:
- Mathematische Aufgaben (Addition, Subtraktion, Multiplikation, Division)
- Logikrätsel (Multiple-Choice)
- Punktesystem mit Rundenwiederholung
